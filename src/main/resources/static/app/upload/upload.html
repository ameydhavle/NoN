SQL Query

You can join tables together to produce a single document containing 1:many relationships. You can use column labels to specify a nested XML structure in the document. For example, in a 1:many between "persons" and "addresses", you can do 'SELECT persons.*, address.street as "address/street", address.city as "address/city", etc', which will result in each address being added to the person document as an address element with street and city child elements.
